Localization Walkthrough

This is a step-by-step walkthrough of adding localization to a website that was built using the Owin Framework.

This walkthrough assumes that you have already completed the Website walkthrough. If you have not completed it yet I suggest you at least read through these instructions before continuing with this walkthrough.

The Pages framework is plumbed to support localization, but it does not contain an implementation of the part that provides localized text yet.

If you would like to see this walkthrough completed please contact the authors. Thanks.
