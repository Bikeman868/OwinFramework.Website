The Pages Repository

This GitHub repository contains a Visual Studio solution that compiles the middleware that implements the Pages functionallity. This functionallity is feature comparable with Microsoft Katana or Microsoft Core but has a very different design with the goal of being an open architecture as well as community owned.

The projects in this repository are all in the Pages Functional Area. Please read that section for overview documentation on how to use these packages.

Note that you can view the readme file from this repository clicking on the 'Readme' link in the header of this page.
