Module Elements

Module elements define a module name and an asset deployment mechanism for all of the page elements that use this module.

Adding a Module to a Package




    If you want to define a module within your package (which is generally a
    good idea if your package contains any Css or Javascript assets), you can
    use the BuildUpPackage method of the fluent builder.
    This technique is very similar to all of the other BuildUpXxxx 
    methods that you use to build your package.
